# stock-ohclv-dataset
This dataset contains historical Open, High, Low, Close, and Volume (OHLCV) stock market data for Apple Inc. (AAPL), Microsoft Corporation (MSFT), and NVIDIA Corporation (NVDA) from 1 January 2015 to 22 October 2025. It is well suited for financial analysis, stock price forecasting, algorithmic trading research, and machine learning or data science projects focused on time-series market data.
